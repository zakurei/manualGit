#簡單 Git 教學

##製作動機
***
雖然 Git 已經是版本控制，及專案管理方面很重要的一個工具，
但是在網路上的中文教學方面，對於新手卻苦於不夠系統的教學很少，
雖然有許多網站，或是個人網誌有教學，但是常常都是片面的指令，卻沒有完整的、系統的教學。
希望能透過這個網站讓欲學習 Git ，這個強大又方便的專案管理軟體的初學者能夠透過簡單的練習小程式，
來達到快速上手的目的。
***

##簡單 Git 教學

本教學包含：

1. 版本控制說明 <br>
2. Git 結構簡介 <br>
3. Git 基本指令教學 <br>
4. Github 教學 <br>
5. Git 簡單操作練習程式 <br>
6. 網路上一些教學資源<br>
